## 프로젝트 소개

**My Insight**는 실시간 데이터 분석 대시보드 웹 애플리케이션입니다. 사용자가 키워드나 데이터를 업로드하면 AI 기반으로 감정 분석, 트렌드 예측 등을 실시간으로 처리하고 시각화하여 인사이트를 제공합니다.

- **주요 기능**:
    - 데이터 업로드 및 키워드 입력
    - 실시간 분석 (감정 분석, 트렌드 추이)
    - 인터랙티브 대시보드 (차트, 그래프)
    - 다양한 키워드 커버리지 지원

이 프로젝트는 2025년 데이터 트렌드를 반영해 개발 중이며, 초기 단계이므로 기능이 점차 확장될 예정입니다.
